# Visión por Computadora (Python)

Repositorio con material, ejemplos y notebooks del curso de Visión por Computadora impartido por el Dr. Raúl Enrique Sánchez Yañez.

## Descripción
Este repositorio reúne código, notebooks y recursos utilizados en el curso de Visión por Computadora. Está orientado a estudiantes y desarrolladores que desean aprender conceptos y técnicas prácticas empleando Python y bibliotecas comunes (OpenCV, NumPy, matplotlib, scikit-image, etc.).

Curso impartido por: **Dr. Raúl Enrique Sánchez Yañez**

## Contenido del curso
- Fundamentos de procesamiento de imágenes
- Operaciones puntuales y aritméticas
- Filtros y detección de bordes
- Transformaciones geométricas
- Segmentación y umbralizado
- Detección y descripción de características (SIFT, SURF, ORB)
- Correspondencia y emparejamiento de puntos
- Reconocimiento y clasificación básicos
- Aplicaciones prácticas y proyectos finales

## Requisitos
- Python 3.8+ (recomendado)
- pip
- Jupyter Notebook / JupyterLab (opcional, pero recomendado)

## Instalación
1. Clona el repositorio:
   git clone https://github.com/Cinthia-hub/Visionporcomputadora-python.git

2. Entra al directorio del proyecto:
   cd Visionporcomputadora-python

3. Crea y activa un entorno virtual (opcional pero recomendado):
   python -m venv .venv
   - Windows: .venv\Scripts\activate
   - macOS/Linux: source .venv/bin/activate

4. Instala dependencias:
   pip install -r requirements.txt

O instala las librerías principales:
   pip install opencv-python numpy matplotlib scikit-image jupyter

## Uso
- Abrir y ejecutar los notebooks:
  jupyter notebook
  o
  jupyter lab

- Ejecuta los scripts de ejemplo desde la línea de comandos:
  python ejemplos/mi_ejemplo.py
